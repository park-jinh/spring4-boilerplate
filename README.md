<div contenteditable="true" translate="no" class="ProseMirror" aria-label="Canvas 편집기" aria-busy="false"><h1>🚀 Spring Boot 4.0 API Boilerplate</h1><p>Java 21 및 Spring Boot 4.0 환경에 최적화된 고성능, 고보안 API 서버 표준 보일러플레이트(Boilerplate)입니다.</p><h2>🛠 기술 스택 (Tech Stack)</h2><ul><li><p><strong>언어(Language):</strong> Java 21 (Eclipse Temurin)</p></li><li><p><strong>프레임워크(Framework):</strong> Spring Boot 4.0.3</p></li><li><p><strong>빌드 도구(Build Tool):</strong> Gradle 8.x</p></li><li><p><strong>핵심 라이브러리:</strong></p><ul><li><p><strong>Lombok / Validation</strong></p></li><li><p><strong>Springdoc OpenAPI 3.0.2</strong> (Swagger)</p></li><li><p><strong>Jasypt 3.0.5</strong> (설정 파일 암호화)</p></li></ul></li></ul><h2>✨ 핵심 기능 (Key Features)</h2><h3>1. 전역 예외 처리 (Global Exception Handling)</h3><ul><li><p><code>@RestControllerAdvice</code>를 통한 중앙 집중식 예외 관리.</p></li><li><p>비즈니스 예외(<code>BusinessException</code>)와 시스템 예외의 명확한 분리.</p></li><li><p>계층화된 커스텀 예외 구조(EntityNotFound, InvalidValue 등)를 통한 세밀한 에러 핸들링.</p></li></ul><h3>2. 공통 응답 규격 (Standard API Response)</h3><ul><li><p>Java 21 <code>record</code>를 활용한 <code>ApiResponse&lt;T&gt;</code> 객체로 모든 응답 통일.</p></li><li><p>성공/실패 시 일관된 JSON 구조를 반환하여 프론트엔드 협업 효율성 증대.</p></li></ul><h3>3. 보안 및 설정 암호화 (Jasypt Security)</h3><ul><li><p><code>application.yml</code> 내 민감 정보(DB, API Key 등)를 <code>ENC()</code> 포맷으로 암호화.</p></li><li><p><strong>마스터 키 주입 우선순위:</strong></p><ol><li><p>OS 환경 변수 (<code>JASYPT_PASSWORD</code>)</p></li><li><p>JVM 시스템 프로퍼티 (<code>-Djasypt.password</code>)</p></li><li><p>설정 파일 내 기본값 (로컬 개발 전용)</p></li></ol></li></ul><h3>4. 로깅 전략 (Logging Strategy)</h3><ul><li><p><strong>Console:</strong> ANSI 색상을 적용하여 가시성 확보.</p></li><li><p><strong>File:</strong> <code>logs/</code> 디렉토리에 일자별 로그 자동 저장.</p></li><li><p><strong>Rolling Policy:</strong> 10MB 단위 분할 보관 및 7일 경과 시 자동 삭제 정책 적용.</p></li></ul><h2>🚀 시작하기 (Getting Started)</h2><h3>1.
